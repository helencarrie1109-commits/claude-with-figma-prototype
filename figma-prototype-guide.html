使用方法

三分钟上手

在 Claude 对话框输入 /figma-prototype 触发技能，告诉 Claude 你要做什么即可。

1

触发技能

在对话框开头输入 /figma-prototype，或者直接说"帮我把这个截图做成 Figma 原型"

2

提供你的 Figma 画布链接

在 Figma 中右键画布 → Copy link，把链接粘贴给 Claude。还没有文件？新建一个空白文件即可。

3

提供截图或网址

直接上传截图，或粘贴一个网页 URL。Claude 会自动分析页面结构和视觉规范，无需任何额外说明。

4

等待构建完成，下载文档

Claude 分批调用 Figma API 逐步构建。完成后打开 Figma 即可看到成果，并收到可下载的 Markdown 设计文档。

使用技巧

让结果更好的三个小贴士

🎯
先新建空白 Figma 文件
Claude 需要一个 Figma 画布链接才能写入。在 Figma 中新建文件后，直接复制浏览器地址栏的 URL 发给 Claude 即可，无需额外操作。
📸
截图比 URL 更精准
如果目标页面需要登录或有动态内容，直接截图比提供 URL 效果更好。Claude 能直接从截图提取色彩和布局细节，还原度更高。
🔁
改完 Figma 随时同步文档
在 Figma 中手动调整设计后，告诉 Claude "帮我同步更新设计文档"，Claude 会读取最新状态并只更新有变化的部分，追加变更记录。
